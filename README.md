# Gerenciador de Produtos

API desenvolvida em ASP.NET Core utilizando Minimal API.

## Tecnologias

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server

## Funcionalidades

- Criar produto
- Listar produtos
- Buscar por Id
- Atualizar produto
- Remover produto
